Control Audio and Video Elements from all tabs from the add-on/extension toolbox

<b>Features:</b>
<ul>
  <li>Control playback</li>
  <li>Focus any site</li>
  <li>Site/media volume Control</li>
  <li>Seek media</li>
  <li>PIP (to be available soon)</li>
  <li>...</li>
</ul>
